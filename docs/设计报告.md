## 摘要

本项目为面向力学教学的桌面虚仿软件，集成教材图解、参数可调仿真、曲线与动画展示及可选 AI 助教说明。平台包含抛体（含空气阻力）、二维装箱碰撞、双摆混沌及通用二维物理沙盒四类模块；数值计算基于 NumPy/SciPy 与 RK4/RK45 流程，界面基于 PyQt5 与 Matplotlib。

---

## (a) 意义与目标定位

中学与大学力学教学中，矢量、能量与微分方程往往依赖静态插图，学生难以建立「参数—轨迹—能量」的整体直觉。本项目定位为 **课堂演示与自主探究** 两用：教师可展示理想与阻力轨迹对照；学生可调初速度、阻力系数、恢复系数、摆长等参数，观察轨迹、能量曲线与相空间行为，强化物理图像与模型认知。

---

## (b) 物理原理

1. **抛体运动**：平面运动状态可取 \(s=[x,y,v_x,v_y]^T\)。理想情形仅受重力；含阻力时采用与速率平方成正比、方向与速度相反的阻力模型，由 \(k,m\) 进入加速度项。对照轨迹可由 SciPy `solve_ivp`（RK45）离线求得；实时仿真在 UI 节拍对应的宏观步长内用 **RK4 微步** 递推。

2. **碰撞模型**：两球在矩形区域内运动，引入恢复系数 \(e\) 描述非弹性碰撞；动能变化可在能量曲线中观察；碰壁可采用弹性反射简化。

3. **双摆**：双自由度耦合非线性系统，可用拉格朗日方程得到角坐标下的耦合方程；仿真采用宏步 + RK4 微步；可开通阻尼项讨论能量耗散；相空间（如下摆 \(\theta_2\)–\(\omega_2\)）用于展示混沌敏感性。

4. **仿真实验室（沙盒）**：二维刚体/弹簧等对象的简化引擎与可视化，用于扩展探究（与课程演示互补）。

---

## (c) 流程与技术实现

- **总体结构**：主窗口 `QStackedWidget` 切换主菜单、三个专题实验页、嵌入的二维物理实验室窗口；专题页采用「教学视图（教材长图 + Markdown/Web 助教）」与「仿真实验室视图（左侧参数 + 右侧 Matplotlib）」两栈切换。

- **数值层**：与 UI 解耦的 `engine/*.py`；抛体分离一次性积分与实时 RK4；双摆、碰撞各自封装步进接口。

- **界面层**：PyQt5；Matplotlib 嵌入 `FigureCanvas`；助教侧可采用 QWebEngine + MathJax 渲染公式（依赖网络或本地策略以实际环境为准）。

- **交互节拍**：仿真采用定时器驱动宏观步进，绘图侧重 `draw_idle` 与坐标轴范围策略以保证流畅度。

## (d) 使用说明（参数可调范围 — 以软件界面为准）

1. **抛体**：初速度、发射角、阻力系数 \(k\)、质量 \(m\)、回放倍速；可启动/暂停/重置；显示理想与有阻力轨迹及能量分量曲线。

2. **碰撞**：两球质量、速率、方向角、恢复系数（非弹性模式）、回放倍速；观察装箱内运动与动能及动能损失曲线。

3. **双摆**：摆长、质量、初始角、阻尼、仿真时长上限、播放倍速；动画、相空间与系统机械能曲线。

4. **仿真实验室**：创建物体类型与属性、播放与重置、数据面板与属性面板等（详见软件内控件）。

5. **设置**：可选配置 DeepSeek API 密钥以启用智能助教对话（无密钥时仍可本地阅读教材与仿真）。

### 演示设计：三实验参数对照与截图建议

为突出「同一初条件下改变系数」的可重复对比，平台在专题实验仿真实验室中提供 **上次运行曲线叠加**（抛体、双摆为轨迹/相轨；碰撞为能量曲线），便于课堂演示与报告插图。

1. **实验一（抛体与阻力）**：固定初速度、发射角与质量，扫描阻力系数 \(k\)。再次点击「开始仿真」时，界面保留 **上一次有阻力轨迹**（灰色虚线）与当前运行（蓝色实线），并与理想无阻力参考并列。报告可配以射程、最大高度或机械能曲线差异的简短结论。

2. **实验二（装箱碰撞）**：两球限制在水平中线运动，多组轨迹在屏幕上重叠度较高，对照应以 **总动能与动能损失（相对初始基准）随时间变化** 为主：固定质量与初速度，改变恢复系数 \(e\)（或弹性/非弹性模式），再次仿真后在同一能量子图中叠加 **上一次** 能量曲线（虚线）。必要时用表格列出 \(e\)、典型碰撞时刻的能量台阶或损失比例，补充定性曲线。

3. **实验三（双摆）**：固定摆长与质量，可扫描阻尼 \(c\) 或微调初始角展示混沌敏感性。再次启动仿真时保留 **上一次的末端拖尾与相空间轨迹**（虚线），与当前运行对比；能量子图仍以本次运行为主，阻尼对照优先观察总机械能衰减快慢。

实验一仿真实验室中点击「重置」会同时清除「上次有阻力」对照线。实验二、三在「重置」时：若已有仿真时间历程，则将其存为灰色对照曲线（图例中标注上一轮典型参数），再按当前面板布置新的初态；便于改变恢复系数、阻尼或摆角后再次「开始仿真」与上一轮对照。

## (e) 结果分析

- **物理合理性**：理想抛体与阻力抛体分离对照；能量曲线在有阻力情况下呈现耗散趋势；碰撞中 \(e<1\) 时动能跃降可观察；双摆在弱阻尼下能量缓慢下降。

- **教学有效性**：同一界面同时给出轨迹、标量时间序列与（双摆）相空间，有利于从「轨道」过渡到「状态空间」概念。

- **可扩展性**：引擎与 UI 分层，可替换阻力模型、增加解析对照或扩展实验列表；沙盒模块可拓展更多物体类型与保存实验场景。

- **局限与改进**：AI 助教依赖外部 API 与网络；复杂三维与连续介质不在范围内；时间步与数值误差需在教学中适当提示；可进一步增加误差估计、实验报告导出与离线公式渲染。
